# ESP8266 Weather Station

Домашняя метеостанция на базе NodeMCU ESP8266.

## Компоненты

- NodeMCU ESP8266
- LCD 1602 I2C дисплей
- DS18B20 датчики температуры
- DS3231 модуль часов реального времени

## Возможности

- Отображение текущего времени
- Показания нескольких датчиков температуры с чередованием
- Синхронизация времени по NTP при старте
- Автономная работа часов от батарейки

## Настройка

1. Скопируй `credentials.h.example` в `include/credentials.h`
2. Заполни WiFi данные в `credentials.h`
3. Настрой параметры в `include/config.h`
4. Собери и залей прошивку через PlatformIO

## Схема подключения

| Компонент | Пин ESP8266 |
|-----------|-------------|
| LCD SDA   | D2          |
| LCD SCL   | D1          |
| LCD VCC   | VIN (5V)    |
| DS18B20   | D4          |
| DS3231 SDA| D2          |
| DS3231 SCL| D1          |
